Kort

An OpenStreetMap Game

  • @KortGame folgen

OpenStreetMap

Kort trägt dazu bei, die OpenStreetMap-Daten zu verbessern. Auf einer Karte werden Aufträge angezeigt, die zu erledigen sind. Sobald genügend übereinstimmende Aufträge gelöst wurden, wird die Lösung eines Auftrags automatisch auf OpenStreetMap eingetragen.

Gamification

Kort nutzt Gamification, um Benutzer zu motivieren, die App zu verwenden. So können durch das Lösen von Aufträgen Punkte (sogenannte Koins) gesammelt werden. Ziel ist es, durch das Sammeln von Koins immer höher in der Rangliste aufzusteigen. Zusätzlich wird ein Spieler für seinen Einsatz auch mit Auszeichnungen belohnt.

App

Kort ist eine native App, die in React Native geschrieben und somit für die meist verbreiteten mobilen Plattformen Android und iOS verfügbar ist.

Kort ist momentan in der Entwicklungsphase und wird im Q2 2017 verfügbar sein. Bleibe mittels GitHub auf dem Laufenden.
